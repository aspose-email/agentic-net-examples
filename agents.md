# Agents

These Aspose.Email samples are maintained by automated agents. On each merge request the following are enforced:
- Root docs: `README.md`, `agents.md`, `index.html` are refreshed.
- Category folders are lowercase with spaces replaced by hyphens; each contains `agent.md` with category notes.
- Sample filenames are generated from the task name, must be meaningful, and stay under 250 characters including `.cs`.
- Examples must use verified Aspose.Email APIs, validate inputs, avoid live calls with placeholder creds/hosts, and wrap network/auth/file operations in safe guards.
